# BigQuery COVID-19 Vaccination Search Analysis

## Project Overview

This project analyzes COVID-19 vaccination search interest using Google BigQuery public datasets and Google Colab. The analysis explores global search trends and focuses on South Africa vaccination interest over time.

## Dataset

Source: Google BigQuery Public Dataset
Table: `bigquery-public-data.covid19_vaccination_search_insights`

## Tools Used

* Google BigQuery (SQL)
* Google Colab
* Python
* Pandas
* Matplotlib

## Analysis Performed

* Loaded dataset from BigQuery
* Explored dataset structure
* Identified top countries by vaccination search interest
* Plotted global comparison chart
* Analyzed South Africa vaccination search trend
* Created visualizations for reporting

## Example Visualizations

* Top countries by vaccination search interest
* South Africa vaccination search trend over time

## Skills Demonstrated

* SQL querying in BigQuery
* Cloud data analysis

* Data visualization
* Working with public datasets
* Portfolio-ready reporting
